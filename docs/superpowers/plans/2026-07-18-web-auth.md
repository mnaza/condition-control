# Web UI Authentication Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Protect the StickC Plus2 web UI and every `/api/*` endpoint with HTTP Basic Auth, opt-in via a password stored in NVS.

**Architecture:** Pure auth logic (base64, header parsing, constant-time compare) lives in `ac-core` and is host-tested. `web.rs` guards every handler with one check, serves 401 + `WWW-Authenticate` on failure, and gains `POST /api/webauth` to set/clear the password. Empty/absent password ⇒ auth disabled. BtnB held at power-on clears the password.

**Tech Stack:** Rust, esp-idf-svc 0.51 (`EspHttpServer`), NVS via existing `net::Store`, vanilla JS in `index.html`.

**Beads epic:** cndition-control-7b0

## Global Constraints

- No new dependencies (hand-rolled ~20-line base64 decoder in `ac-core`).
- Non-blocking firmware: no delays in loop paths (boot-time recovery may block briefly).
- `ac-core` stays free of ESP dependencies; host tests run with `cargo +stable test` in `firmware-stick-rs/ac-core`.
- Firmware build: `cd firmware-stick-rs && source ~/export-esp.sh && cargo build --release`.
- POST bodies are `application/x-www-form-urlencoded` (matches the page's `post()` helper and `form_pairs`), NOT JSON.
- Version bump to 0.3.8 in the final task; commit messages follow repo style, e.g. `Web auth: Basic Auth password for UI + API (v0.3.8)`.

```rust
// --- Web auth ------------------------------------------------------------------

/// Standard-alphabet base64; padded or unpadded. None on malformed input.
pub fn base64_decode(s: &str) -> Option<Vec<u8>> {
    let bytes = s.as_bytes();
    let mut out = Vec::with_capacity(bytes.len() / 4 * 3 + 2);
    let mut acc: u32 = 0;
    let mut bits: u8 = 0;
    let mut pad = 0usize;
    for &c in bytes {
        if c == b'=' {
            pad += 1;
            if pad > 2 {
                return None;
            }
            continue;
        }
        if pad > 0 {
            return None; // data after padding
        }
        let v = match c {
            b'A'..=b'Z' => c - b'A',
            b'a'..=b'z' => c - b'a' + 26,
            b'0'..=b'9' => c - b'0' + 52,
            b'+' => 62,
            b'/' => 63,
            _ => return None,
        };
        acc = (acc << 6) | v as u32;
        bits += 6;
        if bits >= 8 {
            bits -= 8;
            out.push((acc >> bits) as u8);
        }
    }
    if bits >= 6 {
        return None; // a lone extra symbol can't carry a whole byte
    }
    Some(out)
}
```

```rust
/// "Basic <base64(user:pass)>" -> (user, pass). Scheme is case-insensitive.
pub fn parse_basic_auth(header: &str) -> Option<(String, String)> {
    let (scheme, value) = header.trim().split_once(' ')?;
    if !scheme.eq_ignore_ascii_case("basic") {
        return None;
    }
    let text = String::from_utf8(base64_decode(value.trim())?).ok()?;
    let (user, pass) = text.split_once(':')?;
    Some((user.to_string(), pass.to_string()))
}

/// Length leaks; contents don't.
pub fn constant_time_eq(a: &[u8], b: &[u8]) -> bool {
    if a.len() != b.len() {
        return false;
    }
    a.iter().zip(b).fold(0u8, |d, (x, y)| d | (x ^ y)) == 0
}

/// Gate for every web handler. Empty `stored` means auth is disabled.
/// The username in the header is ignored; only the password counts.
pub fn check_password(header: Option<&str>, stored: &str) -> bool {
    if stored.is_empty() {
        return true;
    }
    let Some(h) = header else { return false };
    let Some((_, pass)) = parse_basic_auth(h) else { return false };
    constant_time_eq(pass.as_bytes(), stored.as_bytes())
}
```
